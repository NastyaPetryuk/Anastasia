Ответы на контрольные вопросы:

Что такое «система контроля версий»?

Система контроля версий (VCS) — это программное обеспечение, которое отслеживает изменения в файлах с течением времени. Она позволяет вернуться к любой предыдущей версии файла, сравнивать изменения, совместно работать над проектами и управлять различными ветками разработки.

К какому типу VCS относится Git? В чем главное отличие от других типов?

Git относится к распределенным системам контроля версий (DVCS). Главное отличие от централизованных VCS (например, SVN) заключается в том, что каждый разработчик имеет полную копию репозитория со всей историей изменений. Это позволяет работать автономно и вносить изменения без подключения к центральному серверу.

В чем заключается преимущество использования Git?

  • Распределенная архитектура: Каждый разработчик имеет полную копию репозитория.
  • Поддержка ветвления: Легкое создание и слияние веток для параллельной разработки.
  • Быстродействие: Локальные операции (коммиты, просмотр истории) выполняются быстро.
  • Безопасность: Криптографическая хеш-функция SHA-1 обеспечивает целостность данных.
  • Гибкость: Подходит для различных типов проектов и рабочих процессов.
  • Активное сообщество и поддержка: Большое количество ресурсов, инструментов и библиотек.

Какие команды Git вы знаете?

  • git init: Инициализация нового репозитория Git.
  • git clone: Клонирование существующего репозитория.
  • git add: Добавление файлов в область подготовленных изменений (staging area).
  • git commit: Фиксация изменений в локальном репозитории.
  • git push: Отправка изменений на удаленный репозиторий.
  • git pull: Получение изменений из удаленного репозитория.
  • git branch: Управление ветками.
  • git checkout: Переключение между ветками или восстановление файлов.
  • git merge: Слияние веток.
  • git log: Просмотр истории коммитов.
  • git status: Просмотр состояния репозитория.
  • git diff: Просмотр изменений.

В чем отличие Git и GitHub?

Git — это система контроля версий, программное обеспечение, которое отслеживает изменения в файлах. GitHub — это веб-сервис, хостинг Git-репозиториев в облаке. GitHub предоставляет инструменты для совместной работы, управления проектами и автоматизации рабочих процессов.
